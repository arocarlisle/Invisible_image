# Invisible_image


線上製作【隱形圖片】<br><br>

說明：線上直接就能製作隱形圖片。<br>
原理：只用白色來繪製圖片，透過不同透明度的深淺，讓圖片只要在深色的背景就會顯示出來。<br>
